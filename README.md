# Installation

You can install pipefish for
[Python](https://www.python.org/) via
[pip](https://pypi.org/project/pip/)
from [PyPI](https://pypi.org/).

```
$ pip install pipefish
```




## Prerequisites:
- defusedxml
- docopt
